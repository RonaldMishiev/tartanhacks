# LOCALBOLT EXECUTION GUIDE

## 1. Environment Setup
To start the virtual environment: 
python3 -m venv .venv 

Gather all dependencies:
source .venv/bin/activate
pip install -e .

## 2. Running the Application
# Normal mode (Watch a C++ file)

# Normal mode (Watch a Rust file â€” requires rustc)

# Assembly Help Mode (CLI manual)
